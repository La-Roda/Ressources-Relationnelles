# Ressources-Relationnelles
Projet forum CESI


### Synthaxe des noms de Branches
```
  feature/[nom du ticket]
```

Exemple :
```
  feature/RR-01
```


### Synthaxe des Commits
```
  RR : [message du commit]
  
  [nom du ticket]
```

Exemple :
```
  RR : add logIn and logOut functions
  
  RR-15
```
