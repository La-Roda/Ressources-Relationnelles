# Ressources-Relationnelles
Projet forum CESI


### Syntaxe des noms de Branches
```
  feature/[nom du ticket]
```

Exemple :
```
  feature/RR-01
```


### Syntaxe des Commits
```
  RR : [message du commit]
  
  [nom du ticket]
```

Exemple :
```
  RR : add logIn and logOut functions
  
  RR-15
```
